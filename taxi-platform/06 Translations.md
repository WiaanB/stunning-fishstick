# Translations

Single canonical source file generates both Android `strings.xml` and iOS `Localizable.strings`, to prevent platform drift.

## Confidence levels
- **Higher confidence**: Afrikaans, isiZulu, isiXhosa, Sesotho, Setswana
- **Lower confidence — needs native speaker review before real use**: Sepedi, Xitsonga, siSwati, Tshivenda, isiNdebele
